WebView 电视 v1.12.1：超稳定的开源电视直播App

2025-07-30
 | software
 | reads

软件介绍

大家平时看电视直播的时候，是不是总希望有一款好用又稳定的App呢？

我之前也一直在找，试过不少软件，直到遇到了WebView电视v1.12.1这款App，试了下真感觉还不错，今天就给大家安利一下。

这款App是基于腾讯X5 WebView内核开发的，它通过加载官方直播网页，然后自动识别网页里的video标签，就能实现全屏播放啦，亲测直播观看体验挺流畅的。

它的核心原理挺简单高效的。直接加载官方直播网页，依靠腾讯X5 WebView强大的解析和渲染能力，自动识别video标签并全屏展示。

这样一来，不仅简化了获取直播内容的方式，还保证了直播的稳定性和可靠性。它的优点可多啦。能自动更新频道列表，有多个直播源，要是播放异常还能自动换源或者刷新。而且可以进行应用自定义设置，能适配手机、平板和电视。直接用官网直播链接，稳定得很，还兼容m3u8链接呢，简直是电视直播爱好者的福音。

不过呢，它也有一些小缺点。比起直接加载直播源，它载入频道的时间稍长，得等一会儿。因为用的是网页播放器播放，所以对设备性能有一定要求，并且没办法选择视频清晰度。这里给大家解答两个常见问题。要是在电视机或者盒子上无法自动全屏，一般是设备WebView的问题，等X5内核下载完成，重启App就行了。要是无法自动下载X5内核，在设置 - TBS调试界面内手动安装就行。

要是打开只有CCTV - 1，因为频道列表托管在Github，出现这个问题就检查一下你和Github之间的连接。这个App更新了不少功能。增加了支持x86架构的设备，这对需要用模拟器、x86平板或电视的朋友很有帮助；增加了频道收藏功能，在右侧的频道设置中，点亮“星星”图标就行，频道分组“收藏夹”会显示你收藏过的频道；增加了仅显示收藏分组，在软件设置中，能把“频道列表”切换成“仅收藏”，这对不需要大而全列表的朋友很实用；增加了手动检查更新功能，当有更新时，点击软件设置中的“检查更新”，就能跳转至最新版本的下载页面，这对嫌公众号下载麻烦的朋友很方便。

还修复了在一些设备上运行不久后会闪退的问题、在电视上使用遥控器切换频道分组时左右切换按钮白底偶尔不会消失的问题、使用系统WebView四周会出现黄色方框的问题。

同时也做了不少优化，长按左侧空白处能呼出“退出软件”的界面，能在右侧的频道设置中直接进入软件设置，当系统WebView版本小于78时，在WebView版本号后面会显示“过低”的提示，因为WebView版本过低会导致软件工作异常。

另外，还添加了在Android TV上显示的大图标，移除了上个版本（1.10.9）读取全部应用列表权限，增加了隐藏流量特征开关。也修复了荔枝网无法播放问题和上个版本无法在低版本Android系统上安装的问题。要注意哦，频道列表托管在Github，要是你安装后没有频道列表，或者加载慢，那就是你的网络没办法访问Github。

小结

WebView电视v1.12.1这款App整体来说非常不错，虽然有一些小瑕疵，但优点还是很突出的，能给大家带来比较好的电视直播观看体验，不试白不试啦，有问题评论区说。
