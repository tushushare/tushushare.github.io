前言

在使用电脑的时候，我们经常会遇到一个问题，磁盘空间占用过多，要进行文件清理，需要查看哪些文件夹占用最多，哪些文件最大，但是当你打开文件夹的时候，根本无法看到每个子文件夹下面的文件数有多少，总大小又是多少？虽然可以通过右键，属性来查看，但是需要等待很长的时间，实在是令人捉急啊。

有问题就会有答案（知乎的广告词，还是不错的），今天就给大家分享一个小巧又好用的小工具，专门解决这个问题的，今天的主角是：Folder Size。

介绍

Folder Size 是一个可以在桌面贴边，并显示各子文件夹大小的小工具，2004 年上线，已经存在超过18年，虽然从2013年就没有更新过，但是丝毫不影响实用，目前最新的Windows 11依然能使用，安装包仅 300Kb，小身材大能量。

用法

1. 启动

打开Folder Size后，默认会隐藏到图标栏，内存占用也很低，很不起眼，完全不影响电脑使用。Folder Size运行后，当你打开文件夹的时候，在右上角会有一个快速预览窗口，一眼就能看到当前文件夹下面所有子文件夹的大小。而且还能够根据名字，或者是大小来排序，可以快速的找出哪些子文件夹占用的磁盘空间最多，给你磁盘清理提供最快捷的指南。
