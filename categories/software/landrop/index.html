LANDrop，堪称完美的开源免费文件分享工具！

前言

大家平时在互联网和局域网分享文件都用的什么工具呢？之前介绍过文件共享的网站和微信文件传输助手网页版，在公网情况下，没有其他办法，只能这样，但是在局域网，再选择公网的传输方法，会让你的传输速度大打折扣，最大可能有10倍以上的差距哦。

所以今天给大家介绍一款，目前土叔自认为最好用的文件分享神器，堪称完美，它与之前分享的微信文件助手、文叔叔分享等不同，即使没有互联网连接，只要设备都在同一个局域网就可以正常分享，最重要的是还不限速，他就是：LANDrop。

介绍

该工具是一个跨平台的软件，可以方便快捷的在iphone，ipad，安卓手机、电视、以及windows电脑上使用，除了照片和视频吃外，甚至系统上的任何文件都可以发送到到别的设备上去，真正完美实现的实现了文件互传。

全平台文件互传的这个功能，相信用过苹果系的同学应该都知道，iphone、ipad、macOS等系统上就有一个相当好用隔空投送的功能，但是他却不能和Android的设备分享，同样也不能和windows系统的设备分享。

但是有了今天介绍这款完全免费开源的神器LANdrop后，情况就变得不一样了，用它就可以解决掉你你所有的文件快速分享的烦恼，支持Windows、Android、iOS、macOS、ipad等全平台互传。接下来就详细介绍一下，如何下载、安装和使用。

1. 下载安装

官网下载，打开官网，他默的是英文的，如下图所示，可以看到代码已经开源在github上，有github上的各路大神监督，所以我们可以放心使用，然后找到Downloads，进入下载页面，可以看到有很多不同平台的版本，如ios、android、macOS、windows等，但是，土叔这里只有Android和windows电脑，所以就用这两来演示了，手机APP和电脑版本，打开以后，都不需要进行任何设置，打开就可以直接使用了。

2. 电脑客户端

电脑版本下载完解压后，是一个绿色软件，直接打开即可，电脑主界面是这样的，超级简洁

3. 手机客户端

手机界面是这样的，也同样简洁，在手机上可以看到我的电脑了（DESKTOP-xxx），接下来，就进行照片分享，看下具体效果。

4. 手机向电脑分享

手机上点击右上角的添加按钮，添加照片后，再点击Devices中对应的设备，即可开始发送。

电脑上收到消息后，会推送提示界面，选择“Yes”即可接收照片。

5. 电脑向手机分享

电脑给手机分享照片，方法大致相同，先添加要分享分照片，然后点击发送，会弹出一个设备选择框，选择手机（我的手机是mz-builder6），点击发送即可。

手机上在Receiving栏里可以看到有新消息接收，点绿色的√即可接收。

小结

通过使用发现，传输速度是非常快的，因为他是通过wifi局域网进行传输的，所以它的速度取决于你的WIFI性能。

苹果手机、电脑的操作方法也是一样的，也只需要下载安装就可以使用。

目前能同时实现这种跨全平台，无任何限制的文件分享工具，几乎都是收费的，但是这款工具，是目前发现的最好用的，也是最良心的一款文件分享工具，完全免费开源。

简单易用，不限速，也不限资源，同时还能支持几乎所有主流的操作系统，而且数据只在局域网内进行点了点传输，所以不用担心流量问题，也不会经过任何的第三方服务器，也就意味着他不会被上传到互联网上的任何地方，不会因为网络而导致资料泄露，最大程度保证了隐私安全，非常值得推荐啊。
