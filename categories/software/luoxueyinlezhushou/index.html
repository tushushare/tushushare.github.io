使用方法

1. 主界面介绍

界面布局：洛雪音乐助手的主界面简约大方，主要包含以下部分：

  • 顶部导航栏：包含搜索框、平台选择、设置等功能
  • 左侧菜单栏：包含推荐、热门、最新、飙升等多个栏目
  • 中间内容区：显示歌曲列表、专辑信息等
  • 底部播放栏：显示当前播放歌曲、播放控制按钮

2. 排行榜功能

排行榜浏览：在左侧菜单栏中选择不同的排行榜，如热门榜、新歌榜、飙升榜等，可以查看不同平台的音乐排行榜。

平台切换：在顶部导航栏中可以切换不同的音乐平台，查看各个平台的排行榜内容。

3. 收藏功能

收藏歌曲：在播放歌曲时，点击收藏按钮可以将歌曲添加到收藏列表中。

管理收藏：在左侧菜单栏中选择"我的收藏"，可以查看和管理已收藏的歌曲。

4. 设置功能

基本设置：在顶部导航栏中点击设置按钮，可以进入设置界面，进行以下设置：

  • 下载设置：设置下载路径、音质等
  • 播放设置：设置播放模式、音效等
  • 界面设置：设置主题、语言等
  • 网络设置：设置代理、API接口等

5. 搜索和播放

搜索歌曲：在顶部搜索框中输入歌曲名称、歌手名称等关键词，点击搜索按钮即可搜索歌曲。

播放歌曲：在搜索结果或歌曲列表中，点击歌曲名称即可开始播放。

下载歌曲：在播放歌曲时，点击下载按钮可以下载歌曲到本地。
