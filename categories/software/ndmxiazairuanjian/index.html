前言

介绍

Neat Download Manager是一款免费小巧的多线程下载工具，界面简洁，纯净无广告，外观和IDM几乎一模一样，基本上是高仿。可以进行多线程分段加速下载，直接拉满带宽，跨平台，支持 Windows 和 macOS ，支持HTTP/HTTPS/FTP连接 ，支持HTTP代理和SOCKS代理，支持断点续传，可下载暂停中断的任务继续下载，NDM因为有上面的这些特点，因此速度一点儿也不比迅雷、IDM慢，确实是IDM和迅雷最理想的“替代品”。
