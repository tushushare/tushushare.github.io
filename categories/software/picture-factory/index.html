6. 故障排除与常见问题

软件启动问题

  • 问题：软件无法启动 解决方案：
    • 检查系统是否满足最低要求
    • 以管理员身份运行软件
    • 重新安装软件

  • 问题：无法导入图片 解决方案：

    • 尝试使用"拖放"方式导入

  • 问题：编辑操作无响应 解决方案：
    • 等待软件响应（处理大图片可能需要时间）

    • 重启软件

保存导出问题

  • 问题：无法保存或导出图片 解决方案：
    • 检查保存路径是否存在且可写

    • 尝试使用其他保存格式

性能问题

  • 问题：软件运行缓慢 解决方案：
    • 关闭其他占用系统资源的程序
    • 降低图片分辨率或尺寸
