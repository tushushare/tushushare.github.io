安装与更新

安装方法

方法1：从GitHub下载安装

  1. 访问PowerToys GitHub仓库的发布页面
  2. 下载最新版本的安装包（.exe文件）
  3. 双击安装包，按照提示完成安装

方法2：从Microsoft Store安装

  1. 打开Microsoft Store
  2. 搜索"PowerToys"
  3. 点击"获取"按钮，系统会自动下载并安装

方法3：使用Winget安装

  1. 打开命令提示符或PowerShell
  2. 运行命令：winget install Microsoft.PowerToys
  3. 等待安装完成

更新方法

自动更新：

  • PowerToys默认启用自动更新，会定期检查并安装新版本

手动更新：

  • 打开PowerToys设置

从GitHub更新：

核心工具与功能详解

PowerToys包含多个实用工具，每个工具都有其独特的功能和用途。以下是主要工具的详细介绍：

1. Color Picker（屏幕取色器）

功能介绍

Color Picker是一款轻量的屏幕取色工具，可以快速获取屏幕上任意点的颜色值，支持多种颜色格式。

使用方法

  1. 启动方式：按下 Win+Shift+C 快捷键
  2. 取色操作：移动鼠标到需要取色的位置
  3. 颜色预览：工具会实时显示当前位置的颜色预览
  4. 复制颜色值：点击鼠标左键，颜色值会自动复制到剪贴板
  5. 颜色格式：支持HEX、RGB、HSL等多种格式

应用场景

  • UI设计：获取界面元素的颜色值
  • 前端开发：复制颜色代码到CSS文件
  • 图形设计：匹配已有设计的颜色
  • 办公文档：为PPT、Word等文档获取精确颜色

高级设置

  • 默认颜色格式：可设置默认复制的颜色格式
  • 颜色历史记录：保存最近使用的颜色
  • 自定义快捷键：可修改默认快捷键

2. PowerToys Run（应用启动器）

功能介绍

PowerToys Run是一个快速启动器工具，可以快速启动应用、打开文件、执行命令、计算数值等。

使用方法

  1. 启动方式：按下 Alt+空格 快捷键
  2. 输入关键字：输入应用名称、文件名称或命令
  3. 选择结果：使用方向键选择，按Enter打开
  4. 搜索文件：输入文件路径或名称
  5. 执行命令：输入命令并按Enter执行

主要功能

  • 程序启动：快速启动已安装的应用程序
  • 文件搜索：快速定位并打开文件和文件夹
  • 计算器：直接输入数学表达式进行计算
  • 命令执行：执行CMD、PowerShell等命令
  • 系统设置：快速打开系统设置项
  • 注册表编辑器：快速打开注册表项
  • 网页搜索：直接搜索网页内容

应用场景

  • 快速启动应用：无需在开始菜单中查找
  • 快速打开文件：无需导航文件系统
  • 快速计算：无需打开计算器应用
  • 快速执行命令：无需打开命令提示符

高级设置

  • 搜索引擎：可设置默认搜索引擎
  • 插件支持：支持第三方插件扩展功能
  • 启动速度：优化启动和搜索速度
  • 自定义快捷键：可修改默认快捷键

3. Image Resizer（图片尺寸调整器）

功能介绍

Image Resizer是一个批量调整图片尺寸的工具，可以快速调整一张或多张图片的大小。

使用方法

  1. 启动方式：右键点击图片，选择"调整图片大小"选项
  2. 选择图片：可选择单张或多张图片
  3. 选择尺寸：选择预设尺寸或自定义尺寸
  4. 应用设置：点击"调整大小"按钮
  5. 保存结果：调整后的图片会保存在原文件夹

主要功能

  • 批量调整：同时调整多张图片的尺寸
  • 预设尺寸：内置常用尺寸模板
  • 自定义尺寸：可手动输入宽度和高度
  • 保持纵横比：可选择是否保持图片纵横比
  • 输出格式：可选择输出图片格式

应用场景

  • 社交媒体：调整图片大小以适应不同平台要求
  • 网站开发：优化图片尺寸以提高加载速度
  • 邮件发送：减小图片大小以便于邮件发送
  • 文档插入：调整图片大小以适应文档布局

高级设置

  • 默认设置：可设置默认调整选项
  • 自定义预设：可添加和管理自定义尺寸预设
  • 输出路径：可设置调整后图片的保存路径

4. PowerRename（批量重命名）

功能介绍

PowerRename是一个批量重命名工具，支持使用正则表达式进行高级重命名操作。

使用方法

  1. 启动方式：右键点击文件或文件夹，选择"PowerRename"选项
  2. 选择文件：可选择多个文件进行重命名
  3. 输入规则：在搜索框中输入要替换的文本
  4. 预览结果：实时预览重命名结果
  5. 应用更改：点击"应用"按钮完成重命名

主要功能

  • 批量重命名：同时重命名多个文件
  • 正则表达式：支持使用正则表达式进行高级匹配
  • 实时预览：预览重命名结果，避免错误
  • 大小写敏感：可选择是否区分大小写
  • 文件扩展名：可选择是否包含文件扩展名

应用场景

  • 文件整理：统一命名规则，整理文件
  • 照片管理：批量重命名照片，按日期或事件排序
  • 文档管理：批量重命名文档，按项目或类别分类
  • 代码文件：批量重命名代码文件，遵循命名规范

高级设置

  • 正则表达式：可启用正则表达式模式
  • 替换选项：可设置各种替换选项
  • 文件类型过滤：可过滤特定类型的文件

5. FancyZones（窗口管理器）

功能介绍

FancyZones是一个窗口布局管理器，可以将多个窗口快速布局到桌面的指定区域。

使用方法

  1. 启动方式：打开PowerToys设置，启用FancyZones
  2. 编辑布局：按下 Win+Shift+Z 快捷键编辑布局
  3. 应用布局：拖动窗口时按住 Shift 键，将窗口放到对应区域
  4. 调整窗口：窗口会自动适应区域大小

主要功能

  • 自定义布局：可创建和管理自定义窗口布局
  • 预设布局：内置常用布局模板
  • 多显示器支持：为不同显示器设置不同布局
  • 窗口记忆：记住窗口的位置和大小
  • 快速切换：快速在不同布局之间切换

应用场景

  • 多任务处理：同时处理多个任务，提高工作效率
  • 数据对比：并排查看多个文档或数据
  • 开发工作：同时打开代码编辑器、浏览器和终端
  • 设计工作：同时打开设计工具和参考资料

高级设置

  • 布局编辑器：使用可视化编辑器创建布局
  • 区域行为：可设置窗口在区域中的行为
  • 快捷键：可设置自定义快捷键

功能介绍：增强鼠标功能，提供更多操作选项

6.4 Text Extractor（文本提取器）

功能介绍：从屏幕上的任何位置提取文本

使用方法：

应用场景：

  • 从图片中提取文本

功能介绍：快速预览文件内容，无需打开文件

应用场景：

  • 快速查看图片、文档内容
