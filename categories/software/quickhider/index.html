1.1 安装过程

QuickHider是绿色便携软件，无需安装，解压即可使用。

步骤1：下载压缩包

  • 从官方网站或可信来源下载QuickHider压缩包

步骤2：解压文件

  • 解压后得到QuickHider可执行文件

1.2 启动软件

双击打开软件如下图。

注意事项：

  • 首次运行时，程序会在系统托盘创建图标
  • 主界面不能点击右上角的关闭按钮，否则程序将退出
  • 要隐藏主界面，应使用快捷键Alt+X

2. 基本操作

2.1 窗口隐藏

操作步骤：

  1. 在QuickHider主界面中，切换到"窗口"选项卡
  2. 勾选要隐藏的程序窗口
  3. 点击"隐藏"按钮（如箭头所示）

批量操作：

  • 可以同时勾选多个窗口，点击"隐藏"按钮一次性隐藏
  • 也可以点击"全选"按钮，选择所有窗口后隐藏

2.2 托盘图标隐藏

操作步骤：

  1. 在QuickHider主界面中，切换到"托盘"选项卡
  2. 勾选要隐藏的托盘图标
  3. 点击"隐藏"按钮

注意事项：

  • 隐藏托盘图标不会影响程序的正常运行
  • 隐藏的托盘图标对应的程序仍在后台运行

2.3 任务栏图标隐藏

操作步骤：

  1. 在QuickHider主界面中，切换到"任务栏"选项卡
  2. 勾选要隐藏的任务栏图标
  3. 点击"隐藏"按钮

注意事项：

  • 隐藏任务栏图标不会影响程序的正常运行
  • 隐藏的程序仍在后台运行

2.4 恢复隐藏程序

恢复隐藏的程序也很简单，勾选要恢复的程序，点击恢复按钮即可。

操作步骤：

  1. 在QuickHider主界面中，切换到对应的选项卡（窗口、托盘或任务栏）
  2. 勾选要恢复的程序或图标
  3. 点击"恢复"按钮

批量恢复：

  • 可以同时勾选多个隐藏的程序，点击"恢复"按钮一次性恢复
  • 也可以点击"全选"按钮，选择所有隐藏的程序后恢复

3. 快捷键操作

QuickHider提供了丰富的快捷键，方便用户快速操作。

3.1 默认快捷键

  • 主界面隐藏或显示：默认快捷键是Alt+X
  • 隐藏选中软件：默认快捷键是Alt+H
  • 恢复隐藏的软件：默认快捷键是Alt+R

3.2 自定义快捷键

操作步骤：

  1. 在QuickHider主界面中，点击"设置"按钮
  2. 在设置窗口中，找到"快捷键"选项
  3. 点击要修改的快捷键，按下新的快捷键组合
  4. 点击"确定"保存设置

注意事项：

  • 避免与系统或其他软件的快捷键冲突
  • 选择易于记忆和使用的快捷键组合

3.3 快捷键使用技巧

一般默认操作顺序： 选中软件->隐藏主界面(Alt+X)->隐藏软件(Alt+H)->恢复软件(Alt+R)->摸鱼成功。

紧急情况下的操作：

  1. 当领导突然到来时，按下Alt+X隐藏QuickHider主界面
  2. 按下Alt+H隐藏所有选中的程序
  3. 当领导离开后，按下Alt+R恢复隐藏的程序
  4. 按下Alt+X再次显示QuickHider主界面
