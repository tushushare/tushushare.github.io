我要吹爆的这几款修图工具，免费、简单、实用，绝对的爽！

前言

不知道大家平时处理图片，比如改个尺寸、换个背景、添加水印会用到哪些工具，不过我想大家第一个想到的可能是PS，不得不承认PS最专业，当然是最好的工具，但是却有着很高的操作门槛，只适合高手使用，不适合普通人用。所以今天土叔就分享几个简单免费实用的小工具，帮你解决大部分的图片处理问题。

介绍

1. 改图鸭（网站）

一个完全免费的在线图片处理网站，功能非常丰富，主要有图片压缩、图片转换、图片编辑、抠图换背景、智能证件照、黑白照片上色等等功能。

可以算得上是一个PS在线版，除了上述功能，还有图片改大小、图片加水印、图片旋转、图片裁减、修改分辨率、滤镜、色彩调节、添加文字、叠加、透明度等等功能。

使用方法也很简单，网页上传图片，和PS就差不多了，按照页面上的提示进行操作即可。

2. imagesTool（网站）

又一款图片处理网站，支持图片工具、GIF工具，以及视频工具，支持的功能也非常丰富，包括将pdf、heic、psd、raw、tiff 等超60种以上的格式转换成 jpg、png、webp；给图片添加文字或水印，不透明处理、增加水印边框、增加阴影等，并且速度非常的快，另外还可以进行批量转换。

除了上面的功能外，还有 图片压缩功能，如支持jpg、png、svg、webp、gif、apng等压缩，效果基本上算是无损了，至少肉眼看不出来差别。

3. 简单拼接（Windows）

一款免费图片拼接工具，绿色单文件版，大小只有220K，无需安装，双击exe文件即可打开，主要功能就是拼接啦，虽然体积小巧，但是功能却不少，有横向拼接、纵向拼接、多图拼接等功能，同时还可以设置图片间隔，图片质量。

使用方法非常简单，打开软件，把图片拖入，设置好设置好间隔、质量、调整好图片顺序后，点击【保存拼图】即可完成拼接，是不是很方便！

4. PhotoDemon（Windows）

一款免费、开源、绿色的图片处理工具，算是PS的精简版，支持的功能很齐全，可以裁剪、改分辨率改背景等，通过建新、导入、粘贴图片，左边的工具栏里面有快捷方式，如布局、选择、文字和涂料区，有了这款软件，根本不需要专业的PS知识，也能玩转图片修剪，一些常见的基本功能完全足够了。

小结

好了，今天的修图工具就分享到这里，2款网页版本，2款windows版本，各有各的优点，网页版本无需下载安装即可运行，windows版本小巧精致，可离线使用，大家可以根据自己的需要进行选取使用。
