U-dictionary，有道翻译国际版，会员已到2030年，真香真香！

2022年5月15日
| software
| 阅读

前言

也许大家由于“研究”需要，经常会访问了不少的外文网站，但是比较麻烦的是，外国文字确实看不懂啊，好多软件对相关内容的翻译又极其不准确，甚至还闹出了笑话，所以今天给大家介绍一款好用，正确率很高的翻译软件：U-dictionary。

介绍

U-dictionary，很多人可能以为是国外的软件，其实它就是有道词典的国际版，国际版相对国内版的一大优势在于安装包的体积小且没有广告，**国内版的安装包达到了惊人的162MB，而U-dictionary仅仅只有33.3MB，**多出来的这100多兆怕都是广告吧。

别看这个软件体积不大，但人家可是五脏俱全，日常会用到的功能一个不缺，也没有各种杂七杂八的东西，吊打国内的同类软件。

1. Android端

最基本的功能当然是打字翻译、拍照翻译、以及出国旅游对话翻译，完美解决各种场景下的翻译需求，同时，你也可以下载离线包，即便是无网络的情况下也能实现随用随译，此外，它还可实现屏幕内容时时翻译功能，免除了一边复制一边翻译的操作，大大提升了使用体验，在设置里开启魔法翻译，授予软件悬浮窗权限和无障碍权限，屏幕上就会出现有道词典的悬浮球，把悬浮球拖动到要翻译的地方，即可自动识别并翻译。

今天这款软件已经经过了特殊处理，无需登录即可使用，同时会员功能已开通到2030年。

2. PC端/电脑端

今天也带来了电脑端的特殊版本，绿色版本无需安装即可使用，解压后双击绿化处理程序，就会自动在桌面创建快捷方式，该版本可以登陆账号，与官方客户端相比基本没有差别。登录就是连续包月状态，电脑版的好处就是支持划词翻译功能，不管是任何界面，只要光标轻轻一划，就能自动翻译出结果，是不是非常方便。另外，划词翻译功能还支持以插件的形式安装到浏览器，如果你经常用浏览器观看外语文献，那么这个插件一定可以成为你的得力助手。

小结

有了Android和PC端的搭配使用，畅游网络世界的语言障碍就被完美打破了，接下来就可以尽情地“学习”了，但是比较遗憾的是IOS端暂时没有这种特殊版本，大家将就用吧。
