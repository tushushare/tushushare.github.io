小马模拟器，童年游戏一个都不少，都是曾经的经典！

前言

相信很多小伙伴都听说过小霸王学习机，说是学习机，实际上大家可能都把它当成了游戏机，还记得很小的时候，隔壁邻居家的孩子有一个小霸王，土叔每天都想去他家玩游戏，心里痒痒的很啊，里面非常多的游戏，我玩过的有坦克大战，赛车，魂斗罗，拳皇等等，现在想来这些都是经典游戏啦，满满的都是回忆。可是谁能想到现在，小霸王已经退出了游戏机市场，剩下的只是传说了，基本上都切换到了电脑、主机和手游上了。所以如果要想体验这些小游戏，那就得去找电脑版或者是游戏模拟器才能体验了，因此，今天土叔就给带来了一款手机上的游戏模拟器：小马模拟器。

介绍

小马模拟器目前只有Android版本，里面包含了很多游戏，大概估算了一下有七八千个吧，特别是热门游戏、下载最多，以及精选游戏这三个标签，像曾经比较流行的合金弹头，奥特曼，拳皇，龙珠，恐龙快打，超级玛丽，生化危机，口袋妖怪，街霸等等，另外还有街机、3DS、WII、PSP、PS、DOS、FC、GBA、MD、NGC、SFC、NDS、DC、JAVA…等游戏平台的游戏，游戏平台也是十分的全面，游戏非常的全，非常的完整。我体验了一下马里奥赛车七中文版，虽然画面没有马里奥赛车8那么好看，但是效果还是比较不错，还原度非常高。

1. 游戏安装

选择你想玩的游戏，进入详情页面，然后点击获取即可下载。

2. 启动游戏

下载完成后，可以立即开启，也可以在游戏菜单列表中可以看到你已经下载好的游戏，启动游戏的时候，还需要下载游戏模拟器环境，同时还可能会下载一些其他的运行文件。

3. 运行效果

最后我们来看一下运行效果，我下了个拳皇97但是好像手机有问题运行不起来，所以换成了马里奥赛车7中文版，画面还是比较还原，比较逼真，也能正常的游玩，是否卡顿跟手机有比较大的关系，如果手机配置高，那就很流畅，如果手机配置比较低的话，可能会出现卡顿，屏幕上有模拟左右按键，方向按键，模拟轮盘，还有ABXY键，操作的话跟普通的手机游戏差不多，但是跟真正的3DS比起来还是有一些差别的。

4. 游戏设置

游戏过程中，还可以进行自定义设置，比如玩得不满意的时候，可以重新开始，设置横屏和竖屏，屏幕布局，按键设置，模拟器设置，关闭声音等等。

5. 其他设置

同时还有其他一些配置，比如说游戏管理，模拟器管理，存档管理等等，大家都可以根据自己的需要进行设置。

小结

在体验过程中，除了打开APP的时候有几秒的开屏广告以外，整个操作过程当中没有任何的广告。游戏真的是非常非常多，得到令人惊叹，我相信这些游戏肯定够大家玩好一阵子了。
