前言

平日里，电脑使用过程中，特别是Windows系统的电脑（MacOS自动忽略），你是否也会遇到这样的情况用的好好的，突然程序就卡住了，等半天才恢复正常，用不了多久，又卡住了……

当然电脑卡住的原因有很多很多，比如CPU满负荷运转，CPU温度太高，内存被吃光了，显存不够用，显卡温度过高等等，这些问题往后会一一说明，不过今天还是先聊聊内存优化那些事。

早些年，用过XP和Vista的同学应该知道，当年流行系统优化软件是Windows优化大师，他的功能真的很强大，可以让你研究好几天时间，其中就有内存自动优化这个功能，不过可惜的是Windows优化大师已经停更很多年了。虽然现在的机器，内存已经是按G来算了，但是软件也不甘示弱啊，动不动就占用几个G，尤其是Chrome等一众浏览器，钉钉办公软件等等，所以内存不够用的问题如今依然存在，最近，土叔又开始寻找内存优化的软件了，意外的发现了一款超轻量的Windows内存清理工具：ReduceMemory。

介绍

Reduce Memory 是一款电脑内存清理小工具，如果你不想使用那些花里胡哨的电脑优化软件，如360、电脑管家、鲁大师等等，那么这一款一定要试下，会让你惊喜不已。

1. 主界面

主界面简洁大气，功能简单，点击“优化”即可完成内存优化功能。

2. 开机自启

如果你想在电脑开机后自动启动，后台静默优化，可以在选项中设置下列选项，根据自己需要按下图设置阀值自动清理即可，当然一些不能清理的软件可以自行添加排除在清理之外。

3. 语言设置

语言也可以自行设置，目前支持20+以上语言，默认是跟随系统。

小结

Reduce Memory，专注于内存清理，没有任何多余捆绑功能，小而美，值得拥有。
