前言

说到云盘或者网盘，相信大家首先想到的肯定是百度网盘，当然也有些小伙伴会想到115网盘或者其他的，比如360网盘、腾讯微云等等，但是这些网盘或多或少的都有一些限制，比如说有的下载限速，有的上传限速，还有的空间有限制，总之没有一个完美的解决方案，直到最近，一个良心的网盘横空出世，那就是：阿里云盘。

介绍

阿里云盘是一款由阿里出品的云存储平台，和百度网盘，115网盘功能一样，都是用于在线存储使用，主打速度快、不打扰、够安全、易于分享，你可以在这里存储、管理和探索，尽情打造属于你丰富的数字世界，阿里网盘一出生就不平凡，不仅空间大（免费用户至少可以免费领700G永久空间），而且上传下载还不限速，相比某度网盘难看的吃相，简直是良心到家了。

而且随着越来越多的小伙伴把云盘资料迁移到阿里云盘，阿里云盘的生态越来越完整，资源越来越多，大家搜索起来也越来越方便，再也不用为资源没地方放而发愁了。

1. 主界面

简洁，没有花里胡哨的多与功能，连个广告都没有哦，也许是刚出来，为了拉新吧。

2. 福利社空间领取途径

阿里云盘领取存储空间其实很简单，几乎没有什么门槛，只要按说明完成相应的任务就可以领取到，土叔目前领取了1TB，其中700GB永久空间，其他300GB都有过期时间（1年、4年、5年不等），暂时觉得应该够用了。具体方法为下载阿里云盘手机版，打开福利社版块，号称最高可以领10T容量，但是永久有效的只有700G。

3. 永久空间领取

上传50个文件，获得512G永久空间；发起一次分享，获得128G永久空间；编辑昵称和头像，获得64G永久空间。

4. 其他期限空间领取

做任务获取其他9TB可过期空间，学生的话还有额外1TB（4年有效），大学毕业就过期；邀请好友500G/人，1年有效；开启相册并备份100张照片可获得300G，还有一些其他任务，完成就有不同的奖励。

小结

阿里云盘，界面简洁，易使用，上传下载不限速，就是快，支持超大文件，最高30G，费空间大，永久空间700GB，5年最高可到10TB。
